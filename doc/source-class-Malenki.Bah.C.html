266     /**
267      * Instanciates new character.
268      *
269      * Creates new character object using 3 different ways.
270      *
271      * You can instanciate it using:
272      *
273      *  - a `\Malenki\Bah\N` object. In this case, given object is read to be 
274      *  the UTF-8 code point of the character. So, The Bah Number must be in 
275      *  `[0, 0x10FFFF]` range to be valid.
276      *  - a string-like value having more than one characters must be an HTML entity
277      *  - a single char string-like.
278      * 
279      * Examples:
280      *
281      *     $n = new N(948);
282      *     $c = new C($n);
283      *     echo $c; //''

285      *     $c = new C('&eacute;');
