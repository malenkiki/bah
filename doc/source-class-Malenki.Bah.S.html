 147  * ## Getting parts

 149  * ### Getting character(s)
 150  *
 151  * You can get one or more characters using a method to take one character 
 152  * using its position or by calling magic getters to have a collection of

 156  *
 157  *     $s = new S('azerty');

 160  *     echo $s->charAt(1);
 161  *     // or
 162  *     echo $s->at(1);

 164  * To get a collection of characters as `\malenki\Bah\A` object, use following 
 165  * magic getter:
 166  *
 167  *     $s = new S('azerty');
 168  *     $s->chars; // collection of characters a, z, e, r, t, and  y

 172  *
 173  * You have many ways to get substrings, you can explode the string, take range

 176  * #### Explode everything!
 177  *
 178  * You have two different ways to explode your string into simple manner.
 179  *
 180  *  - By using a given regexp as separator
 181  *  - By cutting each N characters.
 182  *
 183  * So, for each way, you get a `\Malenki\Bah\A` object having many 
 184  * `\Malenki\Bah\S` objects.
 185  *
 186  * Using a regular expression is easy, you have 3 brother methods to do that,

 190  *     $s = new S('1940-06-18 20:00:00');

 192  *     // or
 193  *     echo $s->split('/[\s:-]/')->join(', ');
 194  *     // or
 195  *     echo $s->cut('/[\s:-]/')->join(', ');

 197  * To split the string each N characters, use this:
 198  *
 199  *     $s = new S('abcdefghijklmnopqrstuvwxyz');

 245  * ### Adding new line
 246  *
 247  * You could add LF (`"\n"`) or CR (`"\r"`) or CRLF (`"\r\n"`) quickly using magic gettersâ€¦ `S::$n` and `S::$r` and orthersâ€¦
 248  *
 249  *     $s = new S('Something');

 259  *     $s = new S('Something');
 260  *     echo $s->n(); // print "Something\n"
 261  *     echo $s->n(true); // print "Something\n"
 262  *     echo $s->n(false); // print "\nSomething"
 263  *     echo $s->r(); // print "Something\r"
 264  *     echo $s->r(true); // print "Something\r"
 265  *     echo $s->r(false); // print "\rSomething"

 282  * ## Removing some parts

 284  * ### Stripping
 285  *
 286  * You can strip left and right sides (`S::strip`), just left side 
 287  * (`S::lstrip`) or just right side (`S::rstrip`).
 288  *
 289  * You can strip string either by using methods or by using magic getters.
 290  *
 291  * By using methods, you can change default character to strip. If you use 
 292  * magic getter or methods without arguments, then stripping feature removes 
 293  * only white space. To remove other characters, you must provide them using 
 294  * array, string, `\Malenki\Bah\S` or `\MAlenki\Bah\A` objects:
 295  *
 296  *     $s = new S('   I have some white spaces   ');
 297  *     $s->strip;
 298  *     // or
 299  *     $s->strip();
 300  *     $s = new S('~~.~~~I have some custom chars to remove~~~...~~');
 301  *     $s->strip('.~');
 302  *     // or
 303  *     $s->strip(new S('.~'));
 304  *     // or
 305  *     $s->strip(array('.', '~'));
 306  *     // or
 307  *     $s->strip(new A(array('.', '~')));

 314  *
 315  * You can cast to two different types: objects of primitive PHP types.
 316  *
 317  * ### Casting to objects
 318  *
 319  * To cast to object, only `Malenki\Bah\C` and `\Malenki\Bah\N` are available

 321  *
 322  *  - To cast to `\Malenki\Bah\C`, stringâ€™s length must have exactly one character.

 327  *     $s = new S('a');

 335  *     $s = new S('3');
 336  *     $s->to_c; // casts to \Malenki\Bah\C possible
 337  *     $s->to_n; // casts to \Malenki\Bah\N possible

 339  *     $s = new S('3.14');

 519     {
 520         $args = func_get_args();
 521 
 522         if(!is_null($params)){
 523             if($params instanceof A || $params instanceof H){
 524                 $args = array_values($params->array); //beware of H case
 525             } elseif(is_array($params)){
 526                 $args = $params;
 527             } 
 528         }
 529         $str_out = '';
 530 
 531         foreach ($args as $s) {
 532             self::mustBeStringOrScalar($s);
 533             $str_out .= $s;
 534         }
 535 
 536         return new S($str_out);
 537     }
