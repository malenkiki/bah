Overview

What you need

Bah! is coded and tested on PHP 5.3 to 5.5. So running Bah! on PHP prior to version 5.3 is not good idea.

One PHP extension is mandatory by Bah! Without it, you cannot use it. This extension is multibyte string support aka mbstring.

You must have PHP extension Intl for very few features into S and C classes.

About arguments…

If method has one or many arguments, then, with Bah, you can use either primitive form or Bah equivalent.


use \Malenki\Bah\S;
use \Malenki\Bah\N;
$s = new S('Some very long long string…');
$s->wrap(80);
// is the same as:
$s->wrap(new N(80));

Roles

Bah has 5 classes you can use, each one has specific goal:

Class A
Use it in place of simple array, without named indexes.
It replaces code like this: $arr = array('foo', 2, 'bar', $obj);
Class C
This is special. It is not replacement for primitive PHP type, but it is a class to handle character. So, one instance of the class C stands for one UTF-8 character.
Class H
This replaces named indexed array. Use it in place of $arr = array('foo' => 'bar', 'thing' => 42 );
Class N
It must be used to replace all numeric values: integers, float, double…
Class S
Last but not the least, This is used in place of string. Say good bye to echo trim(mb_strtoupper(' Je suis une chaîne !', 'UTF-8')); and say hello to $s = new S('Je suis une chaîne !'); echo $s->upper->strip;

Getting info

Counts

You can get counts of chars or of bytes contained into the string using 3 ways:

  • the count() function (S implements \Countable interface), returns number of characters as primitive integer value
  • the S::length magic getter, returns number of characters as N object
  • the S::bytes magic getter returns an A object, so you can get number of bytes using length magic getter call or this object or calling count() withi this object as argument.

Now, a little example:


$s = new S('Je suis écrit en français.');
// using count() function
var_dump(count($s->bytes));
var_dump(count($s));

// using objects
var_dump($s->bytes->length);
var_dump($s->length);

This previous code sample should produce following output:

int(28)
int(26)
class Malenki\Bah\N#35 (1) {
  public $value =>
  int(28)
}
class Malenki\Bah\N#114 (1) {
  public $value =>
  int(26)
}

Checking whether the string is void

You can use either void or empty magic getter to check whether the string is void or not. This returns boolean:


$s = new S('');
var_dump($s->void);
var_dump($s->empty);

$s = new S('foo');
var_dump($s->void);
var_dump($s->empty);

$s = new S('   ');
var_dump($s->void);
var_dump($s->empty);

Resulting output should be:

bool(true)
bool(true)
bool(false)
bool(false)
bool(false)
bool(false)

LTR, RTL or both?

Some language use other direction while writting words. For example, in english, we write words form left to right (or LTR), and in arabic languages, or in hebrew, we write words from right to left (or RTL).

Sometimes, you can have text mixing both directions. So, \Malenki\Bah\S class allow you to test directionality of the string using some magic getters and their aliases, returning boolean.

Let’s check whether the given string is LTR:


$s = new S('Ceci est du français tout à fait banal.');
var_dump($s->ltr);
var_dump($s->is_ltr);
var_dump($s->is_left_to_right);
var_dump($s->left_to_right);

All previous lines into code sample will return true.

Now, let’s check whether the given string is RTL or not:


$s = new S('أبجد');
var_dump($s->rtl);
var_dump($s->is_rtl);
var_dump($s->is_right_to_left);
var_dump($s->right_to_left);

Again, all lines will produce true.

Now, let’s see how to check whether string has both directions:


$s = new S('Ceci est du français contenant le mot arabe أبجد qui veut dire "abjad".');
var_dump($s->has_mixed_direction);
var_dump($s->mixed_direction); 
var_dump($s->is_ltr_and_rtl); 
var_dump($s->ltr_and_rtl); 
var_dump($s->is_rtl_and_ltr); 
var_dump($s->rtl_and_ltr);

All should be true!

Finger prints

Magic getters for md5 and sha1 are available, and return another S object.


$s = new S('Hello!');
print($s->md5); // '952d2c56d0485958336747bcdd98590d'
print($s->sha1); // '69342c5c39e5ae5f0077aecc32c0f81811fb8193'

Starts or ends with…

You can check whether string start or end by given another string. This is as simple as that following example:


$s = new S('Hello!');
var_dump($s->startsWith('He'));
var_dump($s->endsWith('yo!'));

The first should return true but second should return false.

If you would test more complicated pattern, you must read contents about regular expressions usage into class S.

Removing parts

Stripping

You can strip left and right sides (S::strip), just left side (S::lstrip) or just right side (S::rstrip).

You can strip string either by using methods or by using magic getters.

By using methods, you can change default character to strip. If you use magic getter or methods without arguments, then stripping feature removes only white space. To remove other characters, you must provide them using array, string, S or A objects:


$s = new S('   I have some white spaces   ');
$s->strip;
// or
$s->strip();

$s = new S('~~.~~~I have some custom chars to remove~~~...~~');
$s->strip('.~');
// or
$s->strip(new S('.~'));
// or
$s->strip(array('.', '~'));
// or
$s->strip(new A(array('.', '~')));

Removing

You can remove other part using replace feature you can find in Regexp chapter.

Adding parts

Prepend, append

Prepending or appending piece of string is as simple as this:


$s = new S('World');
echo $s->prepend('Hello ')->append('!');
// or
echo $s->prepend(new S('Hello '))->append(new S('!'));

You must get Hello World! as resulting string.

Inserting string

You can insert inside the string other string.

To insert string, you must have the string or object having __toString() method, and the position as integer (object or primitive type).


$s = new S('Hello!');
echo $s->insert(' World', 5); // print 'Hello World!'
// or
echo $s->insert(new S(' World'), 5);
// or
echo $s->insert(' World', new N(5));
// or
echo $s->insert(new S(' World'), new N(5));

Repeat again and again…

You can repeat the string how many times you wish. You have just to call S::times() method:


$s = new S('Hi!');
echo $s->append(' ')->times(3)->strip;

You should get Hi! Hi! Hi!. Note I used some other features to add/remove spaces too.

Adding new line

You could add "\n" or "\r" quickly using magic getters… S::n and S::r!


$s = new S('Something');
echo $s->n; // print "Something\n"
echo $s->r; // print "Something\r"
echo $s->r->n; // print "Something\r\n"

This shorthand can be called as method taking one boolean argument to place new line character after (default, true) or before the string (false):


$s = new S('Something');
echo $s->n(); // print "Something\n"
echo $s->n(true); // print "Something\n"
echo $s->n(false); // print "\nSomething"
echo $s->r(); // print "Something\r"
echo $s->r(true); // print "Something\r"
echo $s->r(false); // print "\rSomething"

Join several strings together

To concatenate strings and/or objects having __toString() method, then use static method S::concat(). This method takes any number of arguments and returns S object.


echo S::concat('Ceci est', new S(' une '), ' chaîne'); // print 'Ceci est une chaîne'
echo S::concat('Ceci est', new S(' une '), ' chaîne')->upper; // print 'CECI EST UNE CHAÎNE'

If at least one argument is not a string, then an \Exception is thrown.

Extracting parts

Getting substring using sub(), first and last

You can get first or last character of the string using magic getters:


$s = new S('Hello World!');
echo $s->first; // print 'H'
echo $s->last; // print '!'

This two magic getters return class S object, not class C.

Get substring at any position is easy using S::sub() method. This takes 2 arguments: first is the starting point, second is the number of characters to take. Its returned value is a S object.


$s = new S('Hello World!');
echo $s->sub(); // print 'H', takes first chars by default
echo $s->sub(6, 5); // print 'World'
  • An \InvalidArgumentException is thrown if offset is negative.
  • Another \InvalidArgumentException is thrown if limit is less than one.

Spliting string using Regexp

You can split string using some patterns using regexp.


$s = new S('Hello World!');
var_dump($s->explode('/\s/'));
// or
var_dump($s->split('/\s/'));
// or
var_dump($s->cut('/\s/'));

This previous example get \Malenki\Bah\A object having 2 \Malenki\Bah\S object: Hello and World!

As you can see, their are two aliases available. You can use what you want.

Unlike explode() native PHP function, the separator is regex expression.

  • An \InvalidArgumentException is thrown if pattern is not a string (or object having not __toString() method).
  • Another \InvalidArgumentException is thrown if pattern is void.

Getting parts having same size

You can cut string after each X numbers of characters, to get parts having same size (except last, depending string’s size and part's size).

This is provided by S::chunk() method, which takes only one argument, the size, and it returns \Malenki\Bah\A object containing \Malenki\Bah\S objects.


$s = new S('Hello World!');
var_dump($s->chunk(3));

The result of previous example will have 4 parts: 'Hel', 'lo ', 'Wor' and 'ld!'.

An \InvalidArgumentException is throw if chunk's size is not an integer, or not an \malenki\Bah\N object or if this size is less than 1.
