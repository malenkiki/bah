Overview

What you need

Bah! is coded and tested on PHP 5.3 to 5.5. So running Bah! on PHP prior to version 5.3 is not good idea.

One PHP extension is mandatory by Bah! Without it, you cannot use it. This extension is multibyte string support aka mbstring.

You must have PHP extension Intl for very few features into S and C classes.

About arguments…

If method has one or many arguments, then, with Bah, you can use either primitive form or Bah equivalent.


use \Malenki\Bah\S;
use \Malenki\Bah\N;
$s = new S('Some very long long string…');
$s->wrap(80);
// is the same as:
$s->wrap(new N(80));

Roles

Bah has 5 classes you can use, each one has specific goal:

Class A
Use it in place of simple array, without named indexes.
It replaces code like this: $arr = array('foo', 2, 'bar', $obj);
Class C
This is special. It is not replacement for primitive PHP type, but it is a class to handle character. So, one instance of the class C stands for one UTF-8 character.
Class H
This replaces named indexed array. Use it in place of $arr = array('foo' => 'bar', 'thing' => 42 );
Class N
It must be used to replace all numeric values: integers, float, double…
Class S
Last but not the least, This is used in place of string. Say good bye to echo trim(mb_strtoupper(' Je suis une chaîne !', 'UTF-8')); and say hello to $s = new S('Je suis une chaîne !'); echo $s->upper->strip;

Getting info

Counts

You can get counts of chars or of bytes contained into the string using 3 ways:

  • the count() function (S implements \Countable interface), returns number of characters as primitive integer value
  • the S::length magic getter, returns number of characters as N object
  • the S::bytes magic getter returns an A object, so you can get number of bytes using length magic getter call or this object or calling count() withi this object as argument.

Now, a little example:


$s = new S('Je suis écrit en français.');
// using count() function
var_dump(count($s->bytes));
var_dump(count($s));

// using objects
var_dump($s->bytes->length);
var_dump($s->length);

This previous code sample should produce following output:

int(28)
int(26)
class Malenki\Bah\N#35 (1) {
  public $value =>
  int(28)
}
class Malenki\Bah\N#114 (1) {
  public $value =>
  int(26)
}

Checking whether the string is void

You can use either void or empty magic getter to check whether the string is void or not. This returns boolean:


$s = new S('');
var_dump($s->void);
var_dump($s->empty);

$s = new S('foo');
var_dump($s->void);
var_dump($s->empty);

$s = new S('   ');
var_dump($s->void);
var_dump($s->empty);

Resulting output should be:

bool(true)
bool(true)
bool(false)
bool(false)
bool(false)
bool(false)

LTR, RTL or both?

Some language use other direction while writting words. For example, in english, we write words form left to right (or LTR), and in arabic languages, or in hebrew, we write words from right to left (or RTL).

Sometimes, you can have text mixing both directions. So, \Malenki\Bah\S class allow you to test directionality of the string using some magic getters and their aliases, returning boolean.

Let’s check whether the given string is LTR:


$s = new S('Ceci est du français tout à fait banal.');
var_dump($s->ltr);
var_dump($s->is_ltr);
var_dump($s->is_left_to_right);
var_dump($s->left_to_right);

All previous lines into code sample will return true.

Now, let’s check whether the given string is RTL or not:


$s = new S('أبجد');
var_dump($s->rtl);
var_dump($s->is_rtl);
var_dump($s->is_right_to_left);
var_dump($s->right_to_left);

Again, all lines will produce true.

Now, let’s see how to check whether string has both directions:


$s = new S('Ceci est du français contenant le mot arabe أبجد qui veut dire "abjad".');
var_dump($s->has_mixed_direction);
var_dump($s->mixed_direction); 
var_dump($s->is_ltr_and_rtl); 
var_dump($s->ltr_and_rtl); 
var_dump($s->is_rtl_and_ltr); 
var_dump($s->rtl_and_ltr);

All should be true!

Finger prints

Magic getters for md5 and sha1 are available, and return another S object.


$s = new S('Hello!');
print($s->md5); // '952d2c56d0485958336747bcdd98590d'
print($s->sha1); // '69342c5c39e5ae5f0077aecc32c0f81811fb8193'

Starts or ends with…

You can check whether string start or end by given another string. This is as simple as that following example:


$s = new S('Hello!');
var_dump($s->startsWith('He'));
var_dump($s->endsWith('yo!'));

The first should return true but second should return false.

If you would test more complicated pattern, you must read contents about regular expressions usage into class S.
